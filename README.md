# ASM2 - Booking App

Build an Booking App 

### Features:

-   CRUD Home , Room
-   Authentication and Authorization with JWT

## Instructions for install and running the code:

### In Client:

```c
> cd FrontEnd
```

```c
> npm install
```

```c
> npm start
```

### In Admin Dashboard:

```c
> cd Admin
```

```c
> npm install
```

```c
> npm start
```

### In Server

```c
> cd Backend
```

```c
> npm install
```

```c
> npm start
```

## Technology used :

-   Library : ReactJS , Express JS
-   Database : MonggoDB
-   Style: CSS/SCSS
